VM Requirements:
1.	Preferred OS: Linux (Ubuntu) machine
2.	vCPUs: 8
3.	RAM: 32 GB
4.	Root Disk: 100 GB
5.	Data Disk: 512 GB (If possible 1TB)
6.	Connectivity to access both Gitlab and GitHub Cloud from the VM
7.	Tools to install - Git, GitHub CLI, Python (latest version), Docker
8.	Number of VMs: 2 to 3 for parallel execution


Required if GL-Exporter and GEI tools are used (For migrating Issues and PRs):
Need an Azure Blob Storage to store the Gitlab repo archives (tar files) generated by gl-exporter and share with GEI tool for GitHub migration.
This Blob storage should be accessible by the GEI tool.
